## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| generated bundle → Playwright export render | The compiled artifact code is re-executed in headless Chromium at export resolution |
| browser → export route | User-supplied export request (artifact version id, kind) crosses into a server-side render + storage write |
| export bytes → Convex _storage | Rendered raster/PDF/code bytes uploaded to managed storage |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

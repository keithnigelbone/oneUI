- **RC3 — scaling/fit.** The render-ast page is `minHeight:100vh` desktop-width (~1440px) shown 1:1 in a ~360px card region → cropped.

**User decision (Option A — confirmed, authoritative).** Scope the iframe sandbox to the trust level. Default (absent/false) = strict, so existing untrusted executors are unaffected.

# Phase 04.1 Plan 02: Composer Control Strip Summary

Built `ComposerControlStrip` — a controlled brand / sub-brand / artifact-type / output-profile selector strip lifted verbatim from the retired `RequestPanel`, that the chat composer renders in its `leadingInline` slot. Selections lift to the caller via a single `onChange(value)` and write nothing to tldraw; the Phase-02.1 sub-brand `'skip'` gate is preserved exactly so a stale brandId can never crash the canvas.

## What Was Built

- **Task 1 — `ComposerControlStrip.tsx`** (commit `41df85bd`): A controlled component taking `{ value, onChange, disabled? }` and rendering four `Select`s via the `@oneui/ui-internal/components/Select/Select` deep import. Lifted from `RequestPanel`: the `brandOptions` mapping from `useQuery(api.brands.list)`, `ARTIFACT_TYPE_LABELS`/`ARTIFACT_TYPE_OPTIONS`, the `getValidProfilesForType` profile filtering, and the type-change auto-snap. The `isResolvedBrandId` memo + `'skip'` sentinel are lifted verbatim (Pitfall 4). Every `editor.updateShape` write was replaced with an `emit()` → `onChange` lift. Filled the Plan-01 `composerControlStrip.test.tsx` scaffold with 10 tests (selectors, skip-gate ×3, auto-snap ×2, controlled lift ×2).
- **Task 2 — `ComposerControlStrip.module.css`** (commit `5e00006c`): Token-only compact horizontal strip. Field labels use V2 `--Label-S-*` + `--Label-FontWeight-Medium` + `--Typography-Font-Primary`; gaps via `--Spacing-2`/`--Spacing-1`; selector radius `--Shape-3`. No `background` on a raw container (tinting is the host's `<Surface mode="minimal">` job). Focus halo uses `--Surface-Halo-Gap`, never `--Surface-Main` directly.

**2. [Rule 3 - Blocking] UI-SPEC token aliases not emitted in this repo**
- **Found during:** Task 2 CSS authoring
- **Issue:** The UI-SPEC referenced `--Spacing-3XS`, `--Spacing-5XS`, `--Shape-3XS` — these named aliases are not emitted by `@oneui/tokens` (the repo emits numeric f-step tokens: `--Spacing-2`, `--Spacing-1`, `--Shape-3`, etc.). Writing the non-existent aliases would silently fail to resolve.
- **Fix:** Wrote the real f-step numeric tokens the `RequestPanel.module.css` analog uses, mapping by f-step (3XS≈f-4≈`--Spacing-2`, 5XS≈f-6≈`--Spacing-1`, interactive radius≈`--Shape-3`). Documented the mapping in the module header.
- **Files modified:** `ComposerControlStrip.module.css`
- **Commit:** `5e00006c`

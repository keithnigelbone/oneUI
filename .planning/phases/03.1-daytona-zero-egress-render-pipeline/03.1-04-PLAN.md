|----------|-------------|
| Lab server → Daytona sandbox | the self-contained asset (built from untrusted model TSX) is executed only inside the network-blocked sandbox |
| sandbox origin → canvas iframe | the live preview is served from the sandbox origin; the canvas iframe is strict allow-scripts (no same-origin) |
| DAYTONA_API_KEY → render path | the key is read by `new Daytona()` server-side only; never written into bundle/previewState/screenshot |

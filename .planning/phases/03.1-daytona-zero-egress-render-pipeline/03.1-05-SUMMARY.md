## Deviations (gap-closure from live testing — all fixed)

1. **esbuild route 500** — Turbopack bundled esbuild's native binary; fixed via runtime `createRequire` + `serverExternalPackages`.
2. **Missing snapshot tooling** — `image.ts` referenced a "one-time setup step via daytona.snapshot" that did not exist; built `register-snapshot.ts`.
3. **`require('playwright')` vs `npm i -g`** — global install not on Node's require path; fixed with `NODE_PATH="$(npm root -g)"` on the in-box capture command (no snapshot rebuild).
4. Environmental (user side, not code): stale Convex deploy (`convex dev`), wrong deployment, an `EXPERIENCE_LAB_PREVIEW_EXECUTOR` flag, and a `ddtn_`→`dtn_` key typo.

Added `render-diagnostic` / `probe-diagnostic` harnesses that captured the hidden in-box stderr (`executeCommand` returns only stdout) — they pinpointed the playwright-require failure and verified PREV-01/02/04 + A4 directly.

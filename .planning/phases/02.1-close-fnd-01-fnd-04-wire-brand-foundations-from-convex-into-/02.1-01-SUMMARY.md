## Accomplishments
- Defined the `FoundationsLoaderInput` interface (`brandId` + optional `subBrandConfigId`) and the `FoundationsLoader` type alias (`= (input) => Promise<BrandFoundations | null>`), mirroring the `__setCompileImpl` DI idiom.
- Extended `RunExperienceInput` with `foundationsLoader?` and `subBrandConfigId?`; existing fields untouched.
- Rewrote the no-op `resolveStep` so that, when a loader is injected and `brandFoundations` is unset, it awaits the loader (forwarding `subBrandConfigId` via conditional-optional-spread) and writes any non-null result into `ctx.request.brandFoundations`. The `started`/`completed` emit bracket is preserved.
- Pinned A1: the loader (Plan 02) targets the PUBLIC `getBrandOverviewData`, not the `…Internal` twin, because the public backend HTTP client cannot invoke an `internalQuery`.
- Proved the wiring with tests: loader happy-path → `outcome === 'artifact'`; loader-returns-null → still `artifact` via system defaults (D-08); `subBrandConfigId` forwarding; no-loader no-regression; Pitfall-1 `color.config` mapping guard; `applySubBrandAccentsToFoundation` merge (4 sub-brand roles + inherited semantic roles).
- The agents package remains backend-free — `grep -RE "@oneui/convex|ConvexHttpClient" packages/experience-builder-agents/src` returns nothing.

## Decisions Made
- **A1 resolved (loader uses public `getBrandOverviewData`):** the public query and `getBrandOverviewDataInternal` share one handler (`foundations.ts:975-989`), but the public backend HTTP client cannot call an `internalQuery`, so the loader must use the public query. This is the only place D-06's literal `…Internal` naming bends; the resolved data is identical.
- **resolveStep scope kept minimal:** it only populates `brandFoundations`. `planStep`/`generateStep` already consume it through `resolveFoundation`, and the gap branch lives there — no downstream change needed.
- **Null loader is not a gap (D-08):** a slow/failing/empty loader returns `null`; the resolver then resolves via engine system defaults, never a fabricated foundation gap.

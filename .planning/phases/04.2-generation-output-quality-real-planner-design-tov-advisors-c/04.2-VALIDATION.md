## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | vitest (experience-builder packages + apps/platform Lab `__tests__`) |
| **Config file** | workspace vitest config + per-package |

| **Estimated runtime** | quick per-file < 30s; full suite + gates measured at phase gate |

---

## Sampling Rate

- **After every task commit:** Run the affected package's `pnpm vitest run <path>` (quick).
- **After every plan wave:** Run `pnpm test` + `pnpm typecheck` + `pnpm check:literals` (LAB-02 over the Lab route).
- **Before `/gsd-verify-work`:** Full suite green + `pnpm ci:gates` (single-ai, REG-04, GEN-06).
- **Max feedback latency:** 120 seconds.

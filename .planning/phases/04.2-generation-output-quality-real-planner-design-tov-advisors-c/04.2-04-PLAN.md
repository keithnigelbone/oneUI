The value is documented as empirically tunable in one place; a calibration note is recorded in the SUMMARY.

Purpose: Guarantee that generated web-ui/landing artifacts read as genuine product UIs — real nesting, no placeholders, substantive copy — or honestly report a quality gap.
Output: A primary deterministic gate feeding the existing repair loop, a tuned secondary multimodal gate, and a terminal quality-gap path.
